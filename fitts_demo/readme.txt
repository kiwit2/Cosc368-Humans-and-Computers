Demo version of the Fitts Law Experiment from COSC368 Labs
Updated/Compiled for Python 3.4


To Run:
$ ./fitts.py

To Change Settings:
* Open fitts.py
* Modify the  ``` fitts_impl.BLAH = [...] ``` linesas needed



To Compile/Update this Bundle:
* Grab the fitts_src.zip bundle
* Run fitts.py, to generate the .pyc file for fitts_impl.py
* Copy the __pycache__/fitts_impl.[*].pyc  file (where "[*]" will be some string representing the python interpreter version) and put it alongside fitts_impl.py and fitts.py.
* Zip up the fitts.py and fitts_impl.pyc files into "fitts.zip", along with this messsage.
